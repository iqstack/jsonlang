Jsonlang Docs

Jsonlang is maintained as part of the IQStack project. This project is based on a fork of the Google Jsonnet Project. IQStack is maintaing the sync with the origin project but adding additional functionality as part of the standard library for a Configuration Management DSL (Domain Specific Language).
